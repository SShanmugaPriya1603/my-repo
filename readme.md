# 🚀 My Programming Repository

This repository contains a collection of programs and problem-solving implementations written in **Java**.
It serves as a practice hub for improving **Data Structures & Algorithms (DSA)** skills.

---

## 📂 Contents

The repository currently includes implementations of the following problems:

* 🔹 Binary Order (`btinorder.java`)
* 🔹 Post Order Traversal (`btpostorder.java`)
* 🔹 Pre Order Traversal (`btpreorder.java`)
* 🔹 Length of Last Word (`lengthoflastword.java`)
* 🔹 Longest Common Prefix (`longestcommonprefix.java`)
* 🔹 Majority Element (`majorityelement2.java`)
* 🔹 Maximum Subarray (`maxsub.java`)
* 🔹 Palindrome Check (`palindrome.java`)
* 🔹 Plus One (`plusone.java`)
* 🔹 Remove Duplicates (`remdup1.java`)
* 🔹 Reverse String/Number (`reverse.java`)
* 🔹 Target Sum (`targetsum.java`)
* 🔹 Two Sum (`twosum.java`)
* 🔹 Valid Anagram (`validana.java`)

## ▶️ How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/your-repo-name.git
   ```

2. Navigate to the folder:

   ```bash
   cd your-repo-name
   ```

3. Compile and run any Java file:

   ```bash
   javac filename.java
   java filename
   ```
